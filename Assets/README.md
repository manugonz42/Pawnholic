# Assets

This directory contains images and GIFs for the Pawnholic project.

## Contents

- `gameplay.gif` — Gameplay animation
- `menu.png` — Main menu screenshot
- `shop.png` — Shop interface screenshot

## Specifications

| Image | Size | Format | Description |
|-------|------|--------|-------------|
| gameplay.gif | 800×450 | GIF | Gameplay animation (max 5MB) |
| menu.png | 1280×720 | PNG | Main menu view |
| shop.png | 1280×720 | PNG | Shop interface |

## How to Capture

### GIF
1. Open Unity and run the game
2. Start recording with [ScreenToGif](https://www.screentogif.com)
3. Play through an interesting moment
4. Stop recording and edit if needed
5. Save as `gameplay.gif`

### Screenshots
1. Open Unity and run the game
2. Navigate to the screen you want to capture
3. Take a screenshot (Windows: `Win + Shift + S`)
4. Save the image in this directory
5. Update the README.md to reference the new image

## Optimization

- Use GIF format for animations (max 5MB)
- Use PNG format for screenshots
- Keep images under 1MB
- Compress with [TinyPNG](https://tinypng.com) if needed
